# Textile Inventory Management System

## 📌 Project Description
This project is developed to manage textile inventory such as stock, products, and records efficiently.

## 🛠️ Technologies Used
- Java
- Eclipse IDE
- MySQL (if used)
- Maven

## 🚀 Features
- Add product
- Update stock
- Delete product
- View inventory

## ▶️ How to Run
1. Open project in Eclipse
2. Run as Java Application
3. Configure database if needed
